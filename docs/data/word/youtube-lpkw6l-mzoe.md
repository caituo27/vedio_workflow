# 稀土战争：中美贸易战全面升级的背后

大家好，這裡是二爺故事。 就在今天，原本已經偃旗息鼓的中美貿易戰發生了劇變。 川普連著發了兩篇帖子，宣布從現行的所有關稅外，從11月1號起對所有的中國商品加增100%的關稅。 而且川普還說啊，我原本兩週後將在韓國亞太經合組織會議上與習近平主席會面，但是現在似乎沒有理由這樣做了。 實際上就是取消了這個月底和習近平的會談。
川普的帖子發出來之後，美國的股市應聲暴跌。 可以說大家對中美這兩個大國的貿易前景都不樂觀。 那麼現在大家可能會覺得很奇怪啊，明明幾個月前都已經各退一步達成了休戰協議，怎麼現在又突然像小孩子過家家一樣，兩邊都同時翻臉了呢？ 走到了貿易戰全面升級這一步呢？
這次貿易戰升級的起因來自中共官方發出的兩份通知。 第一份是10月9號商務部發出的稀土出口管制的公告，在原來稀土出口管制的基礎上進一步擴大了範圍，把稀土相關的技術、設備、原材料和輔助料，以及重稀土的氧化物化合物永磁材料等等啊，全部納入了管控範圍。 第二份通知是10月10號交通運輸部發布的關於對美船舶收取船舶特別港務費的公告。
這個公告說啊，將從10月14號起對美國擁有運營註冊建造或者懸掛美國國旗的船舶徵收特別港務費。 也就是以後美國的商業船隻只要停靠中國的港口，都要收取每噸400塊的港務費，而且每半年還要調整一次價格，逐年加價。 在大家的印象裡面啊，在過往的中美貿易戰中，中共都是被動出手，那麼這一次為什麼會在這個時間節點主動出手導致川普如此暴怒呢？
因為啊10月14號這個時間點，就是美國將對中國的船隻徵收港務費的起點時間。 在10月3號的時候，美國海關與邊境保護局發布了一個公告，他宣布從10月14號開始，對進入美國港口的中國的船隻包括運營和建造的船隻，以及所有外國建造的汽車運輸船隻徵收額外的費用。 其中對中國實體擁有的船隻每噸徵收50美元，而對中國建造的船舶按每噸18美元或者個每個集裝箱120美元這個價格來收。
美國海關這個公告簡而言之，就是對所有中國註冊或者是中國建造的船隻，你到了美國的港口後都要按照噸位來收費，也就是俗稱的雁過拔毛。 中國的一艘萬噸級的貿易貨輪到了美國後要繳納的港口停靠費，最低也是50萬美元。 這個數額是非常誇張的啊，你運費一下上升這麼多，那麼必然會影響到終端的商品的成本和價格。
美國這一招對中國的影響有多大了？ 非常大。 根據全國最大的船舶經濟公司克拉克森的研究的數據啊，按照噸位排名，中國目前是世界上第六大船旗國，也就是註冊地在中國的貿易船隻總噸位高達9200萬噸。 更要命的是啊，中國也是目前世界上最大的造船國，僅僅是在2024年，中國建造的船舶就佔了全球造船業的53%。
以半以上啊，目前全球水上貿易船舶總數近1/4是中國造的。 根據英國的海市情報公司的估計啊，根據2024年中國船隻停靠美國的數據來估計的話，中國對美國的貿易此後將要增加的成本在170億美元到230億美元之間。 這可不是個小數目啊。 本來中美的貿易在這幾年持續不斷的貿易戰中已經非常脆弱了，這一下子又增加這麼多成本，那麼對於整個貿易產業鏈的打擊是非常大的。
你看浙江義烏賣到美國的那些小商品，本來就賺不了幾個錢啊，你這運費一下漲這麼多，那還怎麼做呢？ 所以中共在這個節骨眼上出台反擊措施呢也就可以理解了。 那麼美國為什麼要對中國的航運業和造船業下如此的重手呢？ 是不是跟川普說的那樣，是為了正興美國本土的造船業呢？
眾所周知啊，美國的造船業曾經輝煌無比，是當之無愧的世界第一。 但是到了現代也就是2024年，美國的造船噸位已經不足世界造船業的1%，可謂光速墜落。 那麼這個大坑是怎麼挖成的呢？ 1941年珍珠港事件爆發之後啊，美國政府啟動了大規模的造船動員計劃，通過聯邦撥款和暫時動員機制，短短幾年就把美國的造船產能推向了世界第一。
到1943年美國的造船廠年產船舶的總噸位高達1940萬噸，幾乎相當於當時全球其他國家造船量的總合。 當時美國的造船能力有多恐怖呢？ 在1943年的巔峰期，美國平均每天下水三艘7000噸級以上的自由輪。 從開工到下水最快的僅需要四天。 這種人查啊，在美國短短的幾年時間就建造了2710艘啊，到二戰結束的時候，美國和建造的各類運輸船艦總噸位超過了5000萬噸，佔全球總造船量的70%以上是絕對的霸主。
但是到了二戰結束之後啊，大多數國家處在經濟的恢復期，新的造船需求啊就驟減了，沒有那麼多了。 美國過剩的造船產能無處安放啊，他就只能把大量的船廠關門歇業。 很多熟練的產業工人和工程師也被迫轉行。 但是啊造船業它是一個週期性很強的行業。 在60年代經濟復甦之後，尤其是歐洲的船舶需求有回升之後啊，美國的造船業卻沒有抓住這個風口。
造船屬於勞動密集型和技術密集型的混合行業啊，人工和材料成本是大頭。 美國戰後進入了高福利高工資的時代，導致美國本土的人工和材料成本居高不下。 而這個期間，日本和韓國通過政府的扶持和技術的革新，兩個國家的造船業突飛猛進。 結果來就是到了1970年代，美國的造船成本比日本高30%到50%，比韓國高60%到80%。 這麼大的成本差距直接讓美國造船業失去了價格的競爭力啊。
日本和韓國也就取代美國成了世界造船業的兩大支柱。 到1980年的時候，美國造船業在全球的比重已經下降到不足5%了，完全喪失了主導權。 更加雪上加霜的是從90年代開始，改革開放後的中國也加入了造船業的競爭，而且憑藉更加廉價的人工和材料，以及中國人一貫吃苦耐勞的牛馬精神，很快就成了造船業牌桌上重量級的選手。 十幾年的時間就反超了日韓，2006年中國造船業的完工量首次超過日本位居世界第二，僅次於韓國。
僅僅是四年之後，也就是2010年，中國的造船完工量接訂單量以及手持訂單量這三大指標全面超過日本和韓國，正式成為全球第一造船大國。 這種情況一直持續到今天。 那麼你可能會覺得奇怪，面對這麼多後來者的競爭，美國人難道就躺平了嗎？眼睜睜的看著別人在自己擅長的製造業的領域賺錢？ 不是美國不想，而是他們被自己挖的坑困住了。
造成我前面說的美國造船業人工和材料居高不下的重要原因，其實就是瓊斯法案。 美國歷史上有很多類似於禁酒令之類的奇葩法律，對行業造成了致命的影響。 在美國造船業衰落的過程中，瓊斯法案就扮演了這個複雜的角色。 上個世紀20年代初的時候，美國的參議員瓊斯，他就提出了一部1920年商船法案也俗稱瓊斯法案。 因為當時英國、法國、德國這些歐洲的老牌國家是造船也是航運業的霸主，美國還喝不上湯。
所以為了保護本土的造船業和運輸業，就推出了這個瓊斯法案。 法案中規定，在美國國內港口之間的海運船舶必須滿足四大條件，第一在美國建造，第二懸掛美國的國旗，第三由美國公司擁有，第四船員必須主要為美國公民。 這四大條件啊，其實就是完全排除了其他國家在美國開展造船業競爭的可能。 但是我們都知道啊，一個行業它如果完全封閉，固然可以保護本土的製造，但是呢也喪失了外來的競爭。
沒有了競爭，他就很難持續的革新和創造，也很難形成規模化的效應，更不要說提高生產效率，降低製造成本了。 美國造船業在二戰中的輝煌，那是因為在暫時體制之下傾全國之力，發揮了美國工業的實力，但是在和平時期沒有了戰爭的需求，你不可能僅僅依靠政府的訂單來發展啊。 所以這個瓊斯法案出來之後適得其反，在二戰後嚴重的束縛了美國造船業的發展。
圍繞著這個法案形成了一個龐大的既得利益的集團，就包括造船廠工會產業工人等等啊，在這個瓊斯法案的保護下，他們不用面臨國際化的競爭，就依靠美國政府的訂單來生存，自然也就沒有擴大規模提高效率的動力，就導致美國的造船成本啊越來越高，在60年代就相繼被日本和韓國超越了。 舉一個很簡單的例子啊，當時一艘標準的成品油輪，在日本和韓國的建造成本如果是5000萬美元的話，那麼在美國這個成本高達1.5億美元，三倍的差價，如果你是假方，你說你會選擇誰呢？
更學讓加霜的是啊，80年代之後美國政府的產業政策重心就轉移到了航空航天高科技與金融，對傳統的重工業的補貼和扶持呢就相對減少了。 本來就已經虛弱不堪的美國造船業更是江河日下，難以抵禦全球的競爭衝擊。 除了幾個靠軍事訂單支撐的大船廠，其他的船廠幾乎都已經倒閉了。 已經沒有可以用於造民用船隻的船廠了。 到2024年美國造船業在全球的比重實際上已經不足0.04啊，簡直可以說是微不足道。
全年大概只有40到50萬噸的這個造船量，而中國是多少呢？ 4800萬噸，是美國的100倍。 在如此懸殊的對比之下啊，川普想靠收港口停靠費，來讓美國的船舶製造業回流，那怎麼可能呢？ 全世界的訂單啊，就算是不給中國，他也是給了日本和韓國呀，怎麼可能回到成本高昂的美國本土呢？ 而且咱們不帶鄙視的說啊，以美國目前造船業的體量，就是給了你訂單，你也完成不了呀。
因為不要說大型船塢這些硬件啊，單單是產業工人和工程師這一項軟件，那麼沒有20年的累積，你就根本談不上復興啊。 我接下來要說啊，中共這次為何往常大不一樣，根本不給川普面子，以極快的速度就出台了反擊的措施。 但是在中共打出來的兩拳中，只有稀土這一拳是最致命的。 而對美國船隻收取停靠費呢，只是象徵意義的反擊。 為什麼我這麼說呢？
前面我說了啊，美國的造船業目前只佔全球造船份額的0.04%，而在商業造船領域幾乎是沒有存在感的。 在全球的航運業中啊掛美國國旗也就是註冊地在美國的船舶，總噸位大概是1170萬噸，排名全球第20位。 而且由於美國對中國貿易長年的逆差，實際上美國的商船到中國來停靠的並不多。 所以中國針對美國船隻收費和美國針對中國船隻收費啊，那是不對等的。 這一拳打出去呢，美國並不會很痛。 中共真正的殺手劍是稀土。
從今年貿易戰開始以來啊，中共就可謂有備而來，把稀土這個中國佔有壟斷性地位的商品發揮到了極致。 那麼為什麼稀土這個我們平時不太關注的東西會如此重要，而中國又是怎麼成了全球的壟斷性供應商呢？ 所謂的稀土啊，是稀土金屬的簡稱，它包含了抗以和南系金屬一共17種金屬元素。 這17種元素因為化學性質很特殊，不可替代，號稱工業維生素。 在現代工業中具有不可替代的重要性，被廣泛的用於新能源、電子、國防、高端製造領域。
我舉幾個例子啊，比如乳譜低這三種金屬啊，他們就是製造稀土永磁體的合心材料，是新能源汽車的驅動電機、風力發電機組，以及高端機器人等等設備的關鍵材料。 而特有似等等稀土，它是用於液晶顯示、光纖通信、催化劑和拋光粉。 稀土呢還用於航空航天和軍工領域有極其重要的戰略作用。 比如導彈的核心部件，以及雷達系統和先進戰機的一些核心部件等等，由於這些部件對於可靠性的要求非常高，稀土材料是沒有其他的替代品的。 說直白一點啊，稀土就是工業上的言，它使用量不多，每一種產品就要那麼一點點，但是你缺了這一點，你就開不了鍋，炒不了菜。
所以全球的高科技製造業對於稀土是高度依賴啊，已經成了一種絕對的戰略資源。 但是和大多數人想的不一樣的是，稀土雖然帶一個西字，但在全球稀土的儲量其實並不少啊。 全球已經探明的稀土儲量高達1.3億噸，按照目前每年39萬噸的產量啊，可以開採300多年了。 而且很多國家都有啊，中國人是儲量第一4400萬噸，其次是巴西的2100萬噸，印度有690萬噸，澳大利亞、越南、俄羅斯包括美國都有。
這麼多國家都有的礦產，為啥中國就成了行業的壟斷者了呢？ 這個稀土元素啊，它因為化學的性質非常活躍，它在自然中是不能獨立的存在的，它都是以各種化合物伴生礦的形式存在的。 這也就造成了你要提取稀土的元素就必須進行化學的脆取和分離。 而這個脆取和分離的過程，技術含量很高，而且會產生極大的環境污染。 實際上啊大多數國家對於稀土的使用量都不高。
如果一個國家沒有產業化的優勢，單單是為了那一點稀土去搞一整個產業鏈的話，那麼成本太高了，太不划算了，也不一定可能搞得起來，那麼中國為為啥就搞起來了呢？ 1950年代的時候啊，中國在內蒙古的白雲惡博發現了大型的西圖礦，從此呢就有了西土產業。 剛開始的時候呢是非常粗放的，也造成了大量的環境污染。 但是改革開放之後啊，隨著全世界對於稀土的需求劇增，中國的稀土開採和分離技術啊也迅速發展，環保措施呢也慢慢就跟上了。
